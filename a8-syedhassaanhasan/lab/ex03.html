2 - Customize

Implement the map of ex3.1 then:

  • Change to use mapbox tiles: 
    L.tileLayer('https://api.mapbox.com/styles/v1/{id}/tiles/{z}/{x}/{y}?access_token={accessToken}', {
  attribution: 'Map data © OpenStreetMap contributors, CC-BY-SA, Imagery ¬© Mapbox',
  maxZoom: 18,
  id: 'mapbox/streets-v11',
  tileSize: 512,
  zoomOffset: -1,
  accessToken: 'your.mapbox.access.token'
}).addTo(map);
  • Set a dark style (see style URL and Mapbox Dark)
  • Center on USC (adjust center and zoom level
  • Add a pin for USC with a popup (see Leaflet Quick Start Guide)
