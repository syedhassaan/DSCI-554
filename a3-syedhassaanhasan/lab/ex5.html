🏋🏽‍♂️ ex5: bar charts with d3 data join and div elements

Use the starter code to create an horizontal bar chart with d3 for the data provided

NB: d3 data join requires the most set-up but scales with the data!

Data 

data = [200, 150];
Starter code 

<!-- Update this preview -->		
<style>
div.bar {
  background-color: orangered;
  margin-bottom: 6px;
  text-align: right;
  padding-right: 5px;
}
</style>

<!-- load d3 library! -->
<script src="../node_modules/d3/dist/d3.min.js"></script>

<div id="chart"></div>

<script>
data = [100];
d3.select('#chart')
  .selectAll('div')
  .data(data)
  .enter()
  .append('div')
  .attr('class', 'bar')
  .style('width', d => d + 'px')  //arrow function notation
  .text((d, i) => { return  d + 'px div'; });  //arrow function notation
</script>
