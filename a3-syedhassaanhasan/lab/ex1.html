1 - Bars with div and inline style attribute

Using the starter code and data create an horizontal bar chart with div elements and inline style attribute

NB: In the starter code units should be specified for HTML elements dimensions as these can be absolute or relative and several options for units are available, e.g., px, in, mm. Also, margin-bottom is used to separate the bars vertically. You can use the inspector under Computed to see how margin, border and padding are defined.

NB: In general, when graphing in the browser you will use units in pixels!

Data 

data = [200, 150];
Starter code 

<!-- starter showing how to create a div of 100px with 6px margin at the bottom and update this preview -->
<div style="margin-bottom: 6px; text-align: right; background-color: orangered; width: 200px;">200px div</div>
<div style="margin-bottom: 6px; text-align: right; background-color: orangered; width: 150px;">150px div</div>
Answer
200px div
150px div

2 - Bars with div and inline style attribute with CSS inheritance

Modify the code of the ex1.1 to use an outer div that is styled with CSS properties common to all the bars.

NB: By default a div will fill the space, setting the background-color on the outer div will color the whole space the outer div occupies!

NB: div does not inherit the border properties.

Starter code 

<!-- Update this preview -->	
<div style="text-align: right; color: #eee;">
  <div style="margin-bottom: 2px; color: white; background-color: orangered; padding-right: 5px; width: 200px">200px div</div>
  <div style="margin-bottom: 2px; color: white; background-color: orangered; padding-right: 5px; width: 150px">150px div</div>
</div>
Answer
200px div
150px div

3 - Bars with div and class attribute

Modify the code of ex1.2 to use the class attribute to style CSS properties common to all the bars.

Starter code 

<!-- Update this preview -->
<style>
div.bar {
  background-color: orangered;
  margin-bottom: 6px;
  text-align: right;	
}
</style>

<div class="bar" style="width: 200px">200px div</div>
<div class="bar" style="width: 150px">150px div</div>
Answer
200px div
150px div
