1 - Scatterplot with SVG circle

Implement a scatterplot for the data shown using the starter code

Data

        [150, 150]];
Starter code 

<!-- Update this preview -->		
<svg width="300px" height="300px">
  <circle cx="10" cy="10" r="2" fill="orangered"></circle>  <!-- circle center is (cx, cy) and not (x, y)! -->
</svg>

2 - Scatterplot with SVG circle and class attribute

Modify the code of ex3.1 to use the class attribute for common CSS properties.

Starter code 

<!-- Update this preview -->		
<style>
circle.dot {
    fill: orangered;
    r: 2px;
}
</style>

<svg width="300px" height="300px">
  <circle class="dot" cx="10" cy="10"></circle>
</svg>
