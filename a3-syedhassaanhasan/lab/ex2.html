1 - Bars with SVG rect and inline style attribute

Using the starter code show the data provided as an horizontal bar chart

Data 

data = [200, 150];
Starter code 

<!-- Starter of bar with text. Update this preview -->
<svg width="300px" height="300px">
  <rect x="10" y="10" width="100" height="30" fill="orangered"></rect>
  <text x="100" y="30" style="font-size: 12px; fill: black; text-anchor: end;">100px bar</text>

  
</svg>

NB: default units for SVG coordinates are in pixels so it is customary not to add px. Style attribute are according to CSS specifications and therefore it is better to specify the units for positions and dimensions not to have surprises!

2 - Bars with SVG rect and inline style attribute with CSS inheritance

Modify the code of ex2.1 to use an outer group g that is styled with CSS properties common to all the bars.

NB: Because text and shapes share style properties, e.g., fill, CSS inheritance on SVG elements is often not practical to use in graphing. Note that in the starter code the stroke is set on the group g even though it does not support those properties (the g has no appearance), however CSS inheritance passes those properties down to its children. If you zoom on the text for the example you will see that the text has an orangered fill and a black stroke. Note also that for the example the style properties on g (an SVG element) can be moved to the svg element (an HTML element) with the same output because these style properties are named the same on text and rect SVG elements.

Starter code 

<!-- Update this preview -->				
<svg width="300px" height="300px">
  <g style="fill: orangered;">
    <rect x="10" y="10" width="100" height="30"></rect>
    <text x="100" y="30" style="font-size: 12px; text-anchor: end; fill: black;">100px bar</text>
  </g>
</svg>

3 - Bars with SVG rect and class attribute

Modify the code of ex2.2 to use the class attribute for all common CSS properties.

Starter code 

<!-- Update this preview -->				
<style>
  rect.bar { /*style for rect with class bar*/
	fill: orangered;
	height: 30px;	
  }
  text.bar { /*style for text with class bar*/
    fill: black;
    font-size: 12px;
    text-anchor: end;
  }
</style>

<svg width="300px" height="300px">
  <rect class="bar" x="10" y="10" width="100"></rect>
  <text class="bar" x="100" y="30">100px bar</text>
</svg>
