  • Download bootstrap 4.5 examples here.
  • Find in the downloaded files the code for this bootstrap dashboard example and open the page to verify it works.

  • Add to the dashboard page this example of responsive bootstrap grid: 
    
<style>
.col,.col-4,.col-8 {  <!-- make the cells visible -->
  background-color: #ccc;
  border: 0.5px solid black;
}
</style>
<div class="container">
  <div class="row">
    <div class="col">col</div>
    <div class="col">col</div>
    <div class="col">col</div>
    <div class="col">col</div>
  </div>
  <div class="row">
    <div class="col-8">col-8</div>
    <div class="col-4">col-4</div>
  </div>
</div>
